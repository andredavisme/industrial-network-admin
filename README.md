# 🏭 Industrial Network Admin

Admin platform for the **207 Analytix Industrial Network** — manage entities, relationships, products, terms, and portals across Lift Table Production, Mechanical Power Transmission, and Conveyor System Integration.

**Live:** [GitHub Pages](https://andredavisme.github.io/industrial-network-admin/)

## Pages

| Page | Description |
|---|---|
| `index.html` | Dashboard — record counts, funnel stats |
| `entities.html` | Add / edit network entities |
| `relationships.html` | Add / edit entity relationships + funnel qualification |
| `products.html` | Add / edit products |
| `terms.html` | Add / edit entity terms |
| `portals.html` | Manage client portal configs |

## Tech
- Pure HTML/JS, no build step
- Supabase JS v2 for live data
- Hosted via GitHub Pages
